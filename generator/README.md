# Генератор раскладок для МШП — Пятнашки

Этот модуль помогает собирать кастомные поля и получать закодированный параметр `p` для запуска игры. Интерфейс предназначен для подготовки раскладок судейской командой МШП и работает полностью в браузере.

## Возможности

- Выбор размера поля от 2×2 до 8×8.
- Быстрое автозаполнение по порядку или случайная решаемая раскладка.
- Ручное редактирование каждой ячейки.
- Автоматическая проверка уникальности чисел и наличия ровно одной пустой клетки.
- Генерация готовой ссылки и шифра `p` для передачи игрокам.

## Как пользоваться

1. Задайте ширину и высоту поля.
2. Выберите режим автозаполнения и нажмите «Обновить поле», чтобы подготовить сетку.
3. Отредактируйте значения ячеек вручную при необходимости.
4. Нажмите «Создать ссылку», чтобы получить актуальный шифр и URL.
5. Скопируйте ссылку или параметр `p` и отправьте игроку.

Если при редактировании допущена ошибка, генератор сохраняет последний валидный результат и показывает подсказку в блоке сообщений.

## Разработка

Генератор — это статическая страница. Для локальной проверки достаточно открыть `generator/index.html` в браузере. Кодировщик/декодировщик раскладок находится в `scripts/puzzle.js` и используется как игрой, так и генератором.
