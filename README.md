# МШП — Пятнашки

Веб-версия классической игры "пятнашки". Выберите раскладку через параметр `p` в адресной строке или используйте готовые ссылки, чтобы сразу приступить к сборке поля.

- Поддерживаются поля от 2×2 до 8×8.
- Интерфейс адаптирован под настольные и мобильные устройства.
- Счётчик ходов и таймер помогают отслеживать прогресс.

Для создания собственных раскладок воспользуйтесь генератором, расположенным в каталоге [`generator/`](generator/).
